# Social Media Management Tool

Internal operations tool for EyeLevel Growth Studio.

## Stack

- Frontend: React, Vite, Tailwind CSS, ShadCN UI, React Query, Zustand
- Backend: Node.js, Express.js, Prisma ORM
- Database: PostgreSQL
- Cache and queue: Redis, BullMQ
- Auth: JWT, bcrypt
- Real-time: Socket.IO

## Structure

- `frontend/` - React app
- `backend/` - API server
